# 🇧🇩 BanglaType Enterprise Edition

**BanglaType** is a powerful, lightweight, and professional-grade Bangla typing software designed for Windows. It provides a seamless, high-performance experience for both casual users and enterprise professionals.

---

## 🚀 Key Features

### 🔹 Advanced Phonetic Layouts
Type Bangla naturally and intuitively! Using the built-in **BanglaType Phonetic** and **Avro Phonetic** layouts.

### 🔹 Multiple Traditional Layouts
Full support for all popular standard layouts like **Jatiyo** and **Khipro**.

### 🔹 Unicode & ANSI Support
Perfect for designers and professionals! Supports the latest **Unicode** standards for web and general typing, as well as **Legacy ANSI** encoding for specialized design software.

### 🔹 Smart Suggestion Bar
An intelligent, context-aware suggestion engine with a built-in dictionary of over **150,000 words** to speed up your communication.

### 🔹 Windows 11 Mica & Acrylic UI
Modern, translucent interface designed to blend perfectly with the latest Windows aesthetics.

### 🔹 Emoji & Symbol Picker
Quick access to emojis and special symbols (like ৳, ৲) directly from the keyboard interface.

### 🔹 Enterprise Grade Stability
- **Low-Level Keyboard Hook**: System-wide input processing.
- **High-DPI Support**: Crystal clear UI on 4K monitors.
- **Single Instance Control**: Prevents multiple background processes.
- **Enterprise Logging**: Built-in logging for stability monitoring.

## 🛠 Installation & Usage

### 📥 Installation
1. Download the latest installer (`BanglaType_Setup_v1.0.0.exe`).
2. Run the setup and follow the professional installation wizard.
3. BanglaType will automatically start with Windows.

### ⌨️ Quick Shortcuts
- **Toggle Language:** Click the **BN/EN** button on the TopBar or use the tray icon.
- **Settings:** Right-click the tray icon to access the Settings Dialog to toggle Mica UI, Suggestion Bar, or Emoji Picker.

---

## 📁 Repository Structure

- `src/engine`: Core Riti input engine (Rust).
- `windows-port`: Native Windows C++ wrapper and UI logic.
- `data`: Keyboard layouts and dictionary files.
- `release`: Production-ready binaries and installers.
