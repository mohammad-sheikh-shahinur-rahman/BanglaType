Overview

BanglaType Keyboard ("the Software") is a free and intelligent Bangla/English Input Method Editor (IME) for Windows, developed by Mohammad Sheikh Shahinur Rahman. This Privacy Policy describes how BanglaType handles information when you use our software.

Our Core Commitment: BanglaType is built on a privacy-first, offline-first philosophy. Your keystrokes, dictionary entries, clipboard data, and personal typing patterns remain on your device and are never sent to external servers unless you explicitly enable an optional online feature.

This policy applies to all versions of BanglaType Keyboard — including the Professional Setup Installer, Standalone Executable, and Portable Editions (x64/x86).

Data Stored Locally on Your Device

The following data is stored only on your local machine to improve your typing experience:

Data Type Purpose Storage
User Dictionary Custom words and spellings you add for phonetic transliteration Local file only
Clipboard History Last 10 copied texts for quick reuse within the app Local memory (session)
App Preferences Theme choice, layout selection, toggle key settings Local config file
Typing Analytics Local word frequency statistics for smarter suggestions Local database
Custom Layouts User-created keyboard layouts (.kbl files) Local file only
03

How We Use Data

All locally stored data is used exclusively to:

  • Provide accurate Bangla phonetic transliteration and text prediction
  • Personalize your typing experience with learned vocabulary and preferences
  • Maintain your chosen theme, keyboard layout, and application settings
  • Generate local typing analytics for smarter word suggestions
Important: We do not sell, share, rent, or trade any user data to third parties. There is no advertising, tracking, or analytics SDK embedded in BanglaType.

05

Optional Online Features

Certain optional features may require an internet connection. These features are clearly labeled and require your explicit consent before activation:

OPT-IN

Google Suggestions API

When enabled, partial typed text may be sent to Google's Recommendation API to provide enhanced word suggestions. This feature is off by default.

Data sent: Partial word/phrase for suggestion lookup
OPT-IN

Gemini AI Formatting

When explicitly enabled, selected text may be processed by Google's Gemini AI for formatting, tone adjustment, or context-aware suggestions.

Data sent: User-selected text for AI processing
OPT-IN

Voice Typing (Speech-to-Text)

Uses a local browser-based portal with Google Speech Recognition APIs. Audio is processed via Google's services when this feature is actively used.

Data sent: Voice audio stream during active speech input
Note: When you use optional online features, the respective third-party service's own privacy policy applies to the data processed by them. You are responsible for reviewing those policies before enabling these features.

Data Security

We take the security of your data seriously. BanglaType implements the following measures:

Local-Only Storage

All user data is stored locally on your Windows device and never transmitted externally

No Background Services

BanglaType does not run hidden background processes or services that collect data

Safe Mode

Launch with --safe flag to bypass advanced hooks for secure debugging

No Analytics SDK

No third-party analytics, crash reporting, or tracking libraries are embedded

08

Children's Privacy

BanglaType Keyboard is a general-purpose typing tool and is not specifically directed at children under the age of 13. Since the software operates offline and does not collect personal information, it is safe for use by individuals of all ages. However, parental guidance is recommended when enabling optional online features for minors.
