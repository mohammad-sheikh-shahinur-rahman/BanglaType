Smart Transliteration

Type Bangla phonetically using English characters (e.g. ami converts to আমি) or utilize standard fixed layouts like National or Borno.

AI Next Word Prediction

Advanced real-time suggestions using offline frequency maps, Google Recommendations API, and optional cloud spellcheck powered by Gemini AI API.

Browser Voice Typing

Click the microphone in the topbar to trigger the Edge/Chrome voice typing system. Speak in Bangla or English to write instantly.

Safe Mode Recovery

Encountering conflicts? Launch with the --safe command-line argument to run a minimal, stable environment without hooks or overlays.
