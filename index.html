Frequently Asked Questions

What is Safe Mode in BanglaType?
If the software encounters keyboard hook conflicts, launch BanglaType.exe with the `--safe` argument. This runs a minimal, hook-free layout engine that lets you type stably.
How does VC++ Redistributable check work?
The new setup installer bundles Microsoft VC++ x86 & x64 runtimes. On startup, it inspects Windows registries, installing them silently if missing. This prevents dll missing crashes.
Can I build custom layouts?
Yes! Open BanglaType Settings -> Custom Layout Builder. The interactive GUI tool allows you to map custom keyboard key alignments and compile them directly to `.kbl` layout files.
How does Speech-to-Text work?
Clicking the mic icon starts a background local http listener and opens a visual speech transcription portal in Chrome or Edge. It transcribes your voice (English & Bangla) and automatically types it into the active text editor.
